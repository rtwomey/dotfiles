= Dot Files

This is a collection of my various config files. Got the idea from Ryan Bates:
https://github.com/ryanb/dotfiles

To install:

  rake install

== Features

* ZSH

  * Uses oh-my-zsh and the "robbyrussel" theme.
  * Macro to "reset-db" for a Rails project in one line

* vimrc and gvimrc Files

  Includes intelligent autocompletion

* gitconfig

  The global gitconfig I use, with a number of helpful aliases
